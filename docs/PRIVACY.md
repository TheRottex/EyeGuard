# Gizlilik Politikası (Taslak)

Bu belge, EyeGuard’ın gizlilik yaklaşımını tanımlamak için bir taslaktır. Uygulamanızın gerçek veri toplama/işleme durumuna göre **doldurmanız** gerekir.

## 1) Toplanan veriler
- [ ] Kişisel veri topluyor mu? (Evet/Hayır)
- [ ] Telemetri / hata raporu gönderiyor mu? (Evet/Hayır)
- [ ] Hesap / giriş var mı? (Evet/Hayır)

## 2) Veri nerede işlenir?
- [ ] Tamamen cihaz içinde (offline)
- [ ] Sunucuya gönderim var (hangi sunucu, hangi amaç?)

## 3) Üçüncü taraflar
- [ ] Kullanılan 3. parti servisler (Analytics, Crashlytics, vb.)

## 4) İletişim
Gizlilikle ilgili sorular için: [E-posta ekleyin]
